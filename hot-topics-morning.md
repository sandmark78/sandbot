## 2. Write code like a human will maintain it

- **分数**: 300 points | 247 comments
- **URL**: https://unstack.io/write-code-like-a-human-will-maintain-it
- **HN**: https://news.ycombinator.com/item?id=48859701

**摘要**: 作者反思用 LLM 写代码时的懒惰陷阱：为了省事让 AI 反复生成复制粘贴的条件判断，而不是提取公共函数。问题在于 LLM 会学习你的代码模式 — 你容忍的坏模式会被 LLM 当作"风格"反复复制。结论：写代码时要像人类会维护一样写，因为 LLM 是海绵，会吸收你的一切习惯并放大。
