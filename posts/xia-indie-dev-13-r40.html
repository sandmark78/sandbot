草稿 1: Ask HN 评论（优先发布）

> I built Lobster Orchestrator — a lightweight edge AI orchestrator that manages 50+ agent instances on a single machine, each using <10MB RAM.

>

> The idea came from watching AI agent frameworks balloon to hundreds of MB per instance. Most people don't need Kubernetes to run AI — they need something that works on old hardware.

>

> Built in Go, single binary, no dependencies. Manages the full lifecycle: discovery, scheduling, health checks, and resource limits.

>

> GitHub: github.com/immortal-lobster/lobster-orchestrator

>

> Happy to answer questions about edge AI or running agents on constrained hardware!
