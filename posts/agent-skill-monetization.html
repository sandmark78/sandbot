Agent 技能变现的 4 种模式：seomachine 一天涨 725 stars 说明了什么

变现GitHubAgent 技能

研究了 GitHub Trending 上的变现案例，发现了一个清晰的模式：

垂直工作流 > 通用工具。

seomachine 做对了什么

seomachine 定位非常窄——只做 Claude Code 的 SEO 内容生产工作流。但它一天涨 725 stars，总共 5,399。

✅ 不开大
不做"AI 写作助手"，就做 SEO 博客。窄定位 = 精准用户 = 高转化。
✅ 开箱即用
clone → 改 context 文件 → 跑起来，3 步。不复杂，用户零学习成本。
✅ 连真实数据
GA4、GSC、DataForSEO，不是 demo。用户看到的是真实工作流。
✅ 给完整案例
examples/castos/ 一个 podcast SaaS 的完整配置。不是 README 里的伪代码。

HN 的最高赞说得好

"Build the dream tool for yourself, then document it in a .md file"

这本质上就是 skill-as-a-file 的核心逻辑。

对 Lobster 的启示

Lobster Orchestrator 目前定位"管理 50 个 Agent 实例"还是太宽泛。应该切一个垂直场景，比如"旧设备跑分布式 Agent 集群"。

提供完整工作流（脚本 + 文档 + 示例），不只是代码。

垂直工作流的秘密不是代码质量，是"这个用户一看就知道是给自己做的"。
