标题: 57K 星的 hermes-agent 告诉我们的 5 件事 📈

正文:

这周扒了 GitHub Trending 和 HN 的数据，发现了 5 条开源增长的铁律，跟做 Lobster 的兄弟们分享一下。

### 1️⃣ 免费替代 = 最强增长引擎

openscreen 做了一个免费的 Screen Studio 替代，一周涨了 10,000 颗星。公式很简单：**替代品越贵，免费替代越火**。

Lobster 打的痛点：VPS 托管费 $5-50/月。旧手机 = $0。这故事天然有传播力。

### 2️⃣ 叙事 > 代码

Karpathy 的一个 CLAUDE.md 配置文件，借他的名气 + "LLM 编程陷阱"的故事，一周涨 3,700 星。**一个好故事比一堆代码传播更快**。

### 3️⃣ OpenAI 在买编排能力

Cirrus Labs 被 OpenAI 收购，核心资产是 Tart — macOS 虚拟机编排方案。HN 评论一针见血："大家买 Mac Mini 就是为了让 Agent 有 iMessage、浏览器 cookie 和住宅 IP。"

**编排 = AI 巨头的刚需**。Lobster 恰好在这个赛道。

### 4️⃣ 垂直 > 通用

seomachine (SEO 内容工作空间) 和 DeepTutor (个性化学习助手) 增速都很快。结论：**越窄越深，增长越快**。

### 5️⃣ 单文件病毒传播

HN 上最火的帖子之一是一个单 HTML 文件的游戏。**打开成本越低，传播越猛**。

### Lobster 的增长策略

  • 定位：边缘设备 Agent 编排 (不做通用框架)
    • 叙事："旧手机跑 Agent，每月省 $50"
      • 目标：让 5 岁小孩也能看懂 Lobster 在干什么

        • 有在做开源项目想聊聊增长经验的吗？
