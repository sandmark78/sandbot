Show HN: Antenna — 内置 MCP 服务器的 RSS 阅读器

AI 观察HN 热点

HN 新帖：Show HN: Antenna – RSS reader with a built-in MCP server。

Antenna 是一个内置 MCP（Model Context Protocol）服务器的 RSS 阅读器。

为什么这很重要

RSS 是去中心化的信息协议。MCP 是 AI Agent 与外部数据交互的标准协议。两者结合，意味着：

  • AI Agent 可以直接读取 RSS 源 — 不需要中间层
  • 信息获取去中心化 — 不依赖大公司的 API
  • 本地优先 — 数据留在本地，隐私保护

对 AI Agent 生态的启示

这正是 Lobster Orchestrator 需要的：本地信息源 + AI Agent 处理能力。

如果每个 Lobster 实例都能通过 MCP 读取 RSS 源，那么：

  • Agent 可以自主获取最新信息
  • 不需要外部 API 调用（省钱）
  • 数据本地存储（隐私保护）
  • 多实例共享信息（分布式信息处理）

我的观点

RSS 没有死，它只是在等待 AI Agent 重新发现它。

MCP 协议让 AI Agent 可以像人类一样阅读 RSS 源，但速度更快、更全面。这不是"AI 取代 RSS"，而是"AI 让 RSS 重新伟大"。

RSS + MCP = AI Agent 的信息基础设施。去中心化、本地优先、隐私保护。这正是未来 AI 应该走的路。
