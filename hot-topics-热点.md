## 话题 2
- 标题: Microsoft Comic Chat is now open source
- 分数: 670 points (146 comments)
- URL: https://opensource.microsoft.com/blog/2026/07/16/microsoft-comic-chat-is-now-open-source/
- GitHub: https://github.com/microsoft/comic-chat
- 详细描述:
  微软将 1996 年的经典聊天客户端 Comic Chat 开源了。这个软件能将 IRC 聊天自动转化为漫画面板，配有插图角色、对话气泡和表情。

  Comic Chat 的历史意义：它是 Comic Sans 字体的第一个真正应用场景。这个由微软排版师 Vincent Connare 在 1994 年设计的字体，因其非正式的手写风格完美匹配了软件的对话气泡。

  在 1990 年代中期，互联网聊天还是滚动的文字墙。Comic Chat 创新性地将参与者展示为插图角色，对话以漫画面板展开，角色会根据输入内容做出表情和手势——如果有人说"我喜欢那个"，角色会指向自己；如果文本暗示愤怒，角色会皱眉或交叉双臂。

  这个项目由微软研究院虚拟世界组的 David "DJ" Kurlancer 构思，是早期互联网视觉化交流的先驱实验，很多我们现在习以为常的在线交流理念（表情、贴纸、GIF、头像）都能追溯到这类实验。

  现在代码已放在 GitHub 上，开发者、历史学家和复古计算爱好者都可以探索。
